# 個人ブログ作成プロジェクト

Next.jsを使って個人ブログを作成中です。
このプロジェクトは学習目的で作られています。

## 使用技術

- [Next.js](https://nextjs.org)
- TypeScript

## 開発環境のセットアップ

```bash
# パッケージのインストール
npm install

# 開発サーバーの起動
npm run dev
```

[http://localhost:3000](http://localhost:3000)にアクセスして確認できます。

## 学習メモ

- `leaning_note/` ディレクトリに学習過程のメモを残しています
- huskyを使ってGitフックを設定中
